Personal Assistant using Python

Introduction

Project developed using the concepts of python.

The main motive of the project was to overcome the drawbacks of cortana on Windows 10.

It is a CLI (NO use of GUI) based Application.

It makes use of various libraries some of the main are speech_recognition, wikipedia, datetime, OS.

The functionalities of this project include:

  1. Opening Youtube, google by just saying its name.
    Sending an E-mail to the desired user.
    Playing a song from your system. Asking questions (which are then answered using wikipedia or wolframe libraries).

A paper published named Personal Assistant Using Python in IJIRCCE, Volume 9, Issue 4, April 2021
